# GameOfLifeD2D
C# implementation of Conway's Game of Life

This project uses Direct2d via SharpDx. In order to recompile this project, install following nu-get packages.
 
-SharpDX
-SharpDX.Desktop
-SharpDX.Direct2D
-SharpDX.Direct3D
-SharpDX.DXGI
-SharpDX.Mathematics

https://www.nuget.org/packages/SharpDX/
